# Verdinha (nome provisório)

Extensão para Chrome e Firefox (em breve) que análisa o texto da página em busca de nomes de candidatos e congressistas e retorna os maiores doadores de campanha.

Nos moldes de:
http://allaregreen.us/

## Server

Dependencias
* Python
	* Flask
	* PyMongo
	* csvkit
* MongoDB
